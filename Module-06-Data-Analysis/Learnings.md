## Learnings from Module 6: Correlation Analysis and Outlier Detection using Excel

1. **Understanding Correlation Analysis**  
   Learned the concept of correlation analysis and how it helps measure the strength and direction of the relationship between two variables.

2. **Correlation Coefficient Range**  
   Understood that the correlation coefficient ranges from **-1 to +1**, where:  
   - **+1** represents a perfect positive relationship  
   - **-1** represents a perfect negative relationship  
   - **0** represents no linear relationship

3. **Enabling Excel Analysis ToolPak**  
   Learned how to enable the **Excel Data Analysis ToolPak**, which provides statistical analysis tools such as correlation.

4. **Generating a Correlation Matrix**  
   Gained knowledge on how to compute a **correlation matrix in Excel** using the Data Analysis feature.

5. **Interpreting Correlation Values**  
   Learned how to interpret correlation values to determine whether relationships between variables are **strong, weak, positive, or negative**.

6. **Dataset Variables Used**  
   Understood that the variables analyzed in the dataset were **new cases, new deaths, and new vaccinations** from COVID-19 data.

7. **Relationship Between Cases and Deaths**  
   Observed that there is a **strong positive correlation between new cases and new deaths**, meaning increases in cases are generally associated with increases in deaths.

8. **Relationship Between Vaccinations and Deaths**  
   Observed that the **correlation between new vaccinations and new deaths is weak**, suggesting a less direct relationship.

9. **Data Visualization Using Scatter Plots**  
   Learned how to visualize relationships between variables using **scatter plots in Excel**.

10. **Using Trendlines**  
    Understood how to **add and format a trendline** in a scatter plot to better interpret the relationship between variables.

11. **Interpreting Trendline Slope**  
    Learned that a **steeper trendline indicates a stronger relationship**, while a **flatter trendline indicates a weaker relationship**.

12. **Importance of Visual Analysis**  
    Gained the insight that **visual analysis complements statistical results** and helps in understanding patterns within the data.

13. **Correlation vs Causation**  
    Understood that **correlation does not imply causation**, meaning that a relationship between variables does not necessarily indicate that one variable causes the other.

14. **Need for Further Analysis**  
    Learned that additional statistical methods such as **regression analysis or time-series analysis** may be required to draw meaningful conclusions from the data.
